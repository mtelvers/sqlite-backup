
# sqlite3-backup

Backup large SQLite databases

```shell
# sqlite3-backup /var/lib/docker/volumes/ocaml-ci_data/_data/var/db/sqlite.db /var/lib/docker/volumes/ocaml-ci_data/_data/var/db/backup.db
Source database is 92360004 KB plus 115106 KB WAL
Backing up 23090002 pages from /var/lib/docker/volumes/ocaml-ci_data/_data/var/db/sqlite.db
Backup up 23090002 pages to /var/lib/docker/volumes/ocaml-ci_data/_data/var/db/backup.db in 170.028848 seconds
```
